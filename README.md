## Running

Make sure that you have newest `Compass` in your system.

```
sudo gem update
```

### Install Dependencies

```
npm install && bower install
```

### Preview the Application

```
grunt serve
```

Browse to the app at `http://localhost:8000`.


## Testing

### Unit Tests

```
grunt test
```
